## Features

| Feature | Description |
|---------|-------------|
| User Authentication | Register/Login with role selection (Student/Instructor) |
| Course Browsing | Filter by category, level, language, price & search |
| Course Details | Preview lessons, quizzes & instructor info |
| Enrollment System | One-click enrollment with progress tracking |
| Video Lessons | Interactive lesson player with completion marking |
| Quizzes | Multiple-choice quizzes with instant scoring |
| Progress Tracking | Visual progress bars & circular charts |
| Certificates | Auto-generated on course completion |
| Student Dashboard | Overview of enrolled courses & stats |
| Instructor Dashboard | Course management panel for instructors |
| Profile Management | Edit name, email & password |
| Responsive Design | Works on mobile, tablet & desktop |

---

## Tech Stack

- *Backend:* Flask (Python)
- *Frontend:* HTML5, CSS3, JavaScript (inline)
- *Styling:* Bootstrap 5 + Custom CSS
- *Data Storage:* JSON files (no database required)
- *Authentication:* Werkzeug password hashing + Flask sessions
- *Icons:* Font Awesome 6
- *Fonts:* Playfair Display + Inter
